# 👨‍💻 Customer Behavior Data Analyst Portfolio Project

This project represents a complete, industry-standard, end-to-end data analytics workflow. It is designed to demonstrate how technical skills in **Python**, **SQL**, and **Power BI** translate into strategic business value.

## 📝 Project Overview
This project transforms raw data into actionable business intelligence. It covers the full lifecycle of data analysis, from initial ingestion and cleaning to executive-level reporting.

## 📊 Dataset
* **Source:** Customer Shopping Behavior Dataset.
* **Key Attributes:** Demographics, purchase history, and subscription status.
* **Goal:** To identify revenue drivers and optimize customer retention strategies.

## 🛠️ Tools Used
* **Python (Pandas/Seaborn):** For data cleaning and Exploratory Data Analysis (EDA).
* **PostgreSQL:** For complex database querying and data segmentation.
* **Power BI:** To create an interactive, minimalist dashboard.
* **Gamma:** For high-impact strategic storytelling and presentation.

## 🚀 Execution Steps
1. **Python EDA & Cleaning:** Loaded raw data and handled missing values/outliers to ensure data integrity.
2. **PostgreSQL Analysis:** Migrated data to a server and executed queries to extract business metrics.
3. **Power BI Visualization:** Developed a dashboard focusing on revenue and customer trends.
4. **Gamma Presentation:** Built a strategic report to bridge technical findings with business goals.

## 📈 Key Results
* **Revenue Identification:** Identified high-value segments contributing significantly to total revenue.
* **Growth Gap:** Pinpointed a **73% non-subscriber gap** for targeted loyalty conversion.
